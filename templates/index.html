Convert .SVG to .PNG

This service converts SVG files to PNG images.

Drop files here or click to upload.

API usage:

Simply send a POST request with the input file added as a
multipart/form-data with the parameter name file and you will
receive the converted file in return as a PNG file.

Based on AI, PDF and EPS to SVG converter by @Tobias Scheepers
itself based on SVG to PNG converter by @steren
